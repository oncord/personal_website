Project Overview

The Spotify Analytics Dashboard is a comprehensive web application that provides users with detailed insights into their music listening habits. Built as a full-stack application, it integrates with Spotify's Web API to fetch user data and presents it through interactive visualizations and analytics.

This project demonstrates advanced web development skills including OAuth2 authentication, real-time data processing, data visualization, and secure session management. The application allows users to explore their musical preferences, track listening trends over time, and compare their habits with others.

Key Features

  • Spotify OAuth2 Authentication

    Secure login system using Spotify's OAuth2 flow with automatic token refresh

  • Interactive Data Visualization

    Dynamic charts and graphs showing listening trends, top tracks, artists, and genres

  • Historical Data Analysis

    Track listening patterns over different time periods (weekly, monthly, yearly)

  • User Comparison System

    Compare personal listening data with other users and global Spotify trends

  • Mood Analysis

    Analyze music mood and audio features to understand listening patterns

  • Persistent Data Storage

    MongoDB integration for storing user sessions, preferences, and analytics data

Spotify OAuth2 Authentication

// OAuth2 authentication flow
const authSpotify = async (req, res) => {
  const scopes = 'user-read-private user-read-email user-top-read user-read-recently-played';
  const authUrl = `https://accounts.spotify.com/authorize?` +
    `response_type=code&` +
    `client_id=${process.env.SPOTIFY_CLIENT_ID}&` +
    `scope=${encodeURIComponent(scopes)}&` +
    `redirect_uri=${encodeURIComponent(process.env.REDIRECT_URI)}&` +
    `state=${state}`;
  
  res.redirect(authUrl);
};

Challenges & Solutions

OAuth2 Token Management

Challenge: Managing Spotify access tokens with automatic refresh and secure storage

Solution: Implemented a robust token management system using JWT for session storage and automatic token refresh using refresh tokens. Created middleware to handle token validation and renewal seamlessly.

Real-time Data Visualization

Challenge: Creating responsive and interactive charts that update with user data

Solution: Integrated Chart.js with React Chart.js 2 to create dynamic visualizations. Implemented custom hooks for data fetching and state management to ensure smooth updates and performance.

Data Persistence & Analytics

Challenge: Storing and processing large amounts of user listening data efficiently

Solution: Designed a MongoDB schema optimized for analytics queries. Implemented scheduled jobs using node-cron to fetch and process user data periodically, ensuring up-to-date analytics without overwhelming the Spotify API.
